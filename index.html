Introduction to JavaScript

JavaScript is a programming language that adds interactivity to web pages.

JavaScript can be used to create dynamic and interactive web pages.

JavaScript is widely used for web development, animation, and creating interactive web applications.

JavaScript is also used for client-side scripting in web browsers, and it can be used to create server-side scripts using Node.js.

How to print Output in JS?

To print output in JavaScript, you can use the console.log() method.

Example:

console.log("Hello, World!");

Output:

Hello, World!

Data Types in JS

JavaScript has different data types:

  • Number
  • String
  • Boolean
  • Undefined
  • Null
  • Symbol (ES6)
  • Object (including Arrays and Functions)

Example:

var x = 10; // Number

Variable Declaration in JS

Variables are used to store data in a memory location.

Syntax:

var variableName = value;

Example:

var name = "John Doe";

You can also declare and initialize variables in one line:

var name = "John Doe", age = 30;

We can also create variables using let & const.

Using let variable

let name = "John Doe";

Using const variable

const name = "John Doe";

Diffences between variable declaration

There are some differences between variable declaration:

  • var: It is function-scoped and hoisted, meaning its value can be accessed before it is declared. It can be reassigned but not redeclared within the same function.
  • let: It is block-scoped and hoisted, meaning its value can be accessed before it is declared. It can be reassigned but not redeclared within the same block.
  • const: It is block-scoped and hoisted, meaning its value can be accessed before it is declared. It also cannot be reassigned.

Operators in JavaScript

JavaScript supports the following operators:

  • Arithmetic Operators: + , - , * , / , %
  • Assignment Operators: = , += , -= , *= , /= , %=
  • Comparison Operators: > , < , >= , <= , == , === ,!= ,!==
  • Logical Operators: && , || , !
  • Bitwise Operators: & , | , ^ , ~ , << , >> , >>>
  • Ternary Operators: condition? value1 : value2
